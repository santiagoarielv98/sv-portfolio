## Contexto

Comercios con miles de SKUs necesitan catálogos extremadamente rápidos y fáciles de publicar para sus equipos comerciales.

## Problema

Los catálogos tradicionales no escalan bien en rendimiento ni se integran con fluidez a canales de venta conversacional.

## Solución propuesta

Implementación de frontend estático con Astro + Tailwind y backend Laravel API-first, optimizando búsquedas y cacheo de contenido dinámico.

## Enfoque técnico

- Estrategia de rendering orientada a performance y SEO.
- Modelo de datos preparado para gran volumen de variantes.
- Integración con mensajería para cerrar ventas desde conversación.

## Resultado esperado

Una experiencia de catálogo más ágil para usuarios finales y una operación comercial más eficiente para equipos internos.
