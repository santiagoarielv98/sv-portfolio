03 // UI Feeling & Estilo 2026

Filosofía "Vercel / Linear"

  • Radios (Border Radius): Medios y elegantes. Usaremos rounded-xl (12px) para contenedores grandes y rounded-lg (8px) para botones/etiquetas. Evita radios excesivos estilo "pill" a menos que sean insignias (badges).
  • Bordes: Cristalinos y sutiles. Usamos bordes translúcidos border-white/10 para simular el reflejo de la luz sobre vidrio oscuro.
  • Sombras: Nada de sombras negras duras. Usamos sombras compuestas (una para el borde interior, otra difusa para la profundidad) shadow-glass y resplandores de color (glows) para interacciones de hover.
